# Firskbites

Firskbites is a simple web application for ordering food and drinks.

> ⚠️ **Status: Archived / No Longer Maintained**
> This project is no longer actively used or maintained. It is kept public for educational purposes only.

## About

Firskbites was built as a learning project to practice web development concepts, including building a basic ordering system for food and beverages.

## Tech Notes

- The database used in this project is **hardcoded** (no real database connection). Data is stored directly in the code rather than in an external database system.
- This makes the project easy to run and explore without needing to set up a database server.

## Project Status

This project is **discontinued** and is not under active development. There are no plans for further updates, bug fixes, or new features.

## Open Source / Learning Purpose

Firskbites is open source and free for anyone who wants to learn web development. Feel free to:

- Clone or fork the repository
- Explore the code structure
- Use it as a reference or starting point for your own learning project
- Modify it however you like for educational purposes

## Disclaimer

This project was created for learning purposes and is **not intended for production use**. Since the database is hardcoded, it does not support real, persistent data storage.
